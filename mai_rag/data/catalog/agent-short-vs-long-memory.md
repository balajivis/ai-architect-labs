# Short vs Long Memory

Agents use two kinds of memory. Short-term memory holds the immediate conversation or task context, while long-term memory persists facts and preferences across sessions so the agent can remember you next time.

## Key ideas
- Short-term memory is the working context for the current task.
- Long-term memory persists beyond a single session.
- Long-term memory stores facts, preferences, and past outcomes.
- Retrieving the right memory at the right time is the real challenge.
- Together they let agents stay coherent now and personalized over time.
