# Multi-Modal Models

Multi-modal models can take in and reason over more than one type of input, such as text, images, audio, or video, often producing outputs across modalities too. They let a single system describe a picture, answer questions about a chart, or pair language with vision.

## Key ideas
- They combine multiple input types (text, image, audio, video) into one model.
- A common pattern is "see and read," such as answering questions about an image.
- They unlock use cases like document understanding, accessibility, and visual search.
- Output can also be multi-modal, for example generating images or speech.
- Quality varies by modality, so each input type still needs its own evaluation.
