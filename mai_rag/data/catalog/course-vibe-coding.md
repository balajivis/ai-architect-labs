# Vibe Coding (AI for Developers)
Vibe Coding, also called "AI for Developers," is a 2-day course on AI-assisted software development. It teaches developers how to code faster and better with modern AI coding tools like Claude Code and Cursor, turning AI into a genuine pair-programming partner across the development workflow.

## What it covers
- Working fluently with AI coding assistants such as Claude Code and Cursor
- Effective patterns for AI-assisted writing, refactoring, and debugging of code
- Driving multi-step coding tasks and reviewing AI-generated work critically
- Integrating AI tools into real development workflows and codebases
- Habits that keep AI-assisted code maintainable and trustworthy

## Who it's for / Format
This course is for developers and engineers who want to dramatically speed up their coding with AI, distinct from the AI Practitioner course that focuses on building AI products. Delivered as a 2-day live workshop and on the self-paced subscription platform, available in the US and India.
