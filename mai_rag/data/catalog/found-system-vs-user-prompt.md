# System vs User Prompt

A system prompt sets the model's overall role, rules, and tone, while a user prompt carries the specific request in the moment. Separating them gives consistent behavior across many different user requests.

## Key ideas
- The system prompt defines persona, rules, and constraints.
- The user prompt is the immediate request.
- Stable instructions live in the system prompt.
- This separation improves consistency and control.
- It is a building block for assistants and agents.
