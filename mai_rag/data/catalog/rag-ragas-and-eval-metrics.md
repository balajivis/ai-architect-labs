# RAGAS and Eval Metrics

RAGAS is a popular framework for evaluating RAG systems, offering metrics that score things like faithfulness, answer relevance, and how well retrieved context supports the answer. Such metrics let teams measure RAG quality systematically instead of by gut feel.

## Key ideas
- Provides RAG-specific metrics for retrieval and generation quality.
- Common dimensions include faithfulness, relevance, and context fit.
- Many metrics use LLM-as-judge to score outputs at scale.
- Enables comparison across pipeline changes and configurations.
- Metrics guide where to invest — better retrieval vs. better prompting.
