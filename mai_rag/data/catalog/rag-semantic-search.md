# Semantic Search

Semantic search finds results by meaning rather than exact keyword match, so a query like "how to reset my login" can surface a doc titled "password recovery steps." It relies on embeddings and vector similarity.

## Key ideas
- Matches intent and meaning, not just literal words.
- Handles synonyms, paraphrases, and related concepts gracefully.
- Powered by embeddings compared via similarity in vector space.
- Often combined with keyword search for the best of both worlds.
- Quality depends on the embedding model and how content was chunked.
