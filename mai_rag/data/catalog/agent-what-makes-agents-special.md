# What Makes Agents Special

A plain language model predicts text. An agent is special because it can take actions in the world, observe what happens, and decide what to do next. That feedback loop is the difference between a system that talks and a system that gets things done.

## Key ideas
- Agents act, not just respond: they call tools, fetch data, and change state.
- They operate in a loop, so they can recover from errors and pursue multi-step goals.
- They can be given autonomy to decide which steps and tools are needed.
- This unlocks use cases like research, automation, and orchestration of other systems.
- The trade-off is unpredictability, which is why evaluation and oversight matter.
