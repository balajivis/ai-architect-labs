# Tools Directory & Cheat Sheets
The Tools Directory and Cheat Sheets are practical Library resources that help learners pick the right AI tools and apply techniques quickly. They make the platform useful in day-to-day work, not just during lessons.

## What it is
- A tools directory pointing to the AI tools worth knowing
- Cheat sheets for fast, practical reference on common tasks
- At-a-glance guidance that speeds up real work
- A complement to deeper course material in The Library
- Resources kept relevant to current AI tooling

## Who it's for / Format
These resources serve every Modern AI Pro learner who wants quick, actionable references and tool recommendations. They live within The Library on the self-paced subscription platform, available in the US and India.
