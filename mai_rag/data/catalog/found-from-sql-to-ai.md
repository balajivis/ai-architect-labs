# From SQL to AI

"From SQL to AI" frames the shift from querying structured databases with exact commands to working with AI systems that handle ambiguous, natural-language requests. It helps people with data backgrounds map familiar concepts onto AI.

## Key ideas
- SQL is precise and deterministic; AI is probabilistic and flexible.
- Natural language replaces rigid query syntax for many tasks.
- AI handles unstructured data that SQL cannot.
- Both still matter; AI complements rather than replaces databases.
- A data background transfers well into AI work.
