# Prompt Engineering Elements

Prompt engineering is the practice of crafting inputs that reliably get good results from a model. Well-structured prompts usually combine clear instructions, relevant context, examples, and a defined output format.

## Key ideas
- Clear instructions and goals improve results.
- Context and examples steer behavior.
- Specifying an output format makes responses usable.
- Iteration and testing beat one-shot guessing.
- Good prompting is a learnable, repeatable skill.
