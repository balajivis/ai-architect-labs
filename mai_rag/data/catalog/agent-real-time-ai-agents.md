# Real-Time AI Agents

Real-time AI agents respond and act with low latency, often while interacting with a live user or streaming data. They prioritize responsiveness, so the experience feels immediate rather than batch-processed.

## Key ideas
- Real-time agents emphasize fast, interactive responses.
- They may stream output as it's produced for a snappier feel.
- Use cases include live chat, voice assistants, and monitoring.
- Latency, streaming, and quick tool calls become design priorities.
- They balance speed against the depth of reasoning per step.
