# Handoffs

A handoff is when one agent passes a task — along with its context — to another agent or to a human better suited to continue. Handoffs are how specialized parts of a system cooperate without one agent trying to do everything.

## Key ideas
- A handoff transfers control and relevant context to the next handler.
- It lets specialists take over the part they're best at.
- Handoffs can go agent-to-agent or agent-to-human.
- Preserving context across the handoff is essential.
- Clean handoffs keep multi-agent systems coherent.
