# Agent-to-Agent Communication

Agent-to-agent communication is how separate agents exchange information, requests, and results. Whether agents share a goal or negotiate one, they need a reliable way to talk so the system stays coherent.

## Key ideas
- Agents exchange messages carrying context, requests, and outputs.
- Clear message formats prevent misunderstanding between agents.
- Communication underpins delegation, handoffs, and coordination.
- It can be direct between two agents or routed through a coordinator.
- Emerging standards aim to make cross-agent communication interoperable.
