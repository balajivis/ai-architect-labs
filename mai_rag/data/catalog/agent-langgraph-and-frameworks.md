# LangGraph and Agent Frameworks

Agent frameworks are libraries that handle the plumbing of building agents — loops, tools, state, and coordination — so builders focus on behavior instead of boilerplate. LangGraph is one popular framework that models agent workflows as a graph of steps.

## Key ideas
- Frameworks provide reusable building blocks for agents.
- LangGraph represents workflows as connected nodes and edges.
- This graph model makes branching, loops, and state explicit.
- Frameworks ease checkpoints, human-in-the-loop, and observability.
- Choosing one depends on the system's complexity and team preferences.
