# The Playground
The Playground is Modern AI Pro's hands-on sandbox where learners experiment without leaving the platform. It combines a code sandbox and a prompt tester so ideas from lessons can be tried immediately and safely.

## What it is
- A code sandbox for experimenting with AI techniques
- A prompt tester for iterating on and comparing prompts
- An immediate, low-friction place to practice what lessons teach
- A bridge between watching Explainers and building real projects
- An always-available space to build intuition through doing

## Who it's for / Format
The Playground is for every Modern AI Pro learner who wants to learn by experimenting, from beginners testing prompts to engineers trying techniques. It is part of the self-paced subscription platform, available in the US and India.
