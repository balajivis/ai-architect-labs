# Vibe Coding (AI-Assisted Development)

Vibe coding describes building software primarily by directing AI tools in natural language, rather than writing every line by hand. It lets people with limited coding background prototype and ship working software faster, while still requiring judgment about correctness and quality.

## Key ideas
- AI assistants generate, edit, and explain code from natural-language intent.
- It lowers the barrier to building, especially for prototypes and internal tools.
- Reviewing, testing, and understanding the output still matters — AI can be wrong.
- It shifts the human role toward specifying intent, verifying, and iterating.
