# Context Window

The context window is the amount of text a model can consider at once, covering both the prompt you send and the response it generates. Once content exceeds this limit, older material must be dropped or summarized.

## Key ideas
- It bounds how much the model can "see" in a single request.
- It includes both input and generated output.
- Larger windows enable longer documents and conversations.
- Filling it fully can raise cost and slow responses.
- Strategies like summarizing or retrieval help work within limits.
