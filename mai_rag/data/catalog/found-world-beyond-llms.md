# The World Beyond LLMs

AI is broader than large language models. Computer vision, speech, recommendation systems, robotics, and classic machine learning all matter, and many real products combine LLMs with these other techniques.

## Key ideas
- LLMs are one part of a much larger AI landscape.
- Vision, speech, and recommendations remain important fields.
- Classic ML still wins for many structured-data problems.
- Strong systems often blend several techniques.
- Knowing the landscape prevents using an LLM for everything.
