# Build Your First Agent

Building your first agent is the leap from a chatbot that talks to a system that acts. You give the model one or two tools and a loop, and it begins deciding what to do, doing it, and reacting to the results.

## Key ideas
- An agent adds tools and a decide-act-observe loop to a chatbot.
- Starting small — one goal, a couple of tools — keeps it understandable.
- The agent chooses when to use a tool versus answer directly.
- Watching its steps helps you learn how it reasons.
- Guardrails keep early agents safe while you experiment.
