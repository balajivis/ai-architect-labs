# Fine-Tuning (LoRA / QLoRA)

Fine-tuning adapts a pre-trained model to a specific domain, style, or task using additional targeted data. Efficient methods like LoRA and QLoRA make this cheaper by adjusting only a small portion of the model.

## Key ideas
- It specializes a general model for a narrower need.
- Efficient methods (LoRA, QLoRA) cut cost and hardware demands.
- It is one option alongside prompting and retrieval.
- It needs quality task data to pay off.
- It is often unnecessary when prompting or RAG suffice.
