# Build Your First Chatbot

A chatbot is the simplest entry point into building with language models: a system that takes a user message and returns a helpful reply. It's where most builders start before moving on to agents that take actions.

## Key ideas
- A chatbot maps user input to a model-generated response.
- A system prompt sets its persona, scope, and tone.
- Conversation context lets it follow a multi-turn dialogue.
- It's a stepping stone toward tool-using agents.
- Good chatbots set clear boundaries on what they will and won't do.
