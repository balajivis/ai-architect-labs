# AI Practitioner vs Vibe Coding — Which One?
Two Modern AI Pro courses are aimed at developers but answer different questions: one teaches you to build AI, the other teaches you to code faster with AI. Choosing correctly depends on your goal.

## What it covers
- AI Practitioner / AI Architect (3 days): build production AI systems — RAG, evals, MCP, trust
- Vibe Coding / AI for Developers (2 days): AI-assisted coding with Claude Code and Cursor
- Build AI products → AI Practitioner; code anything faster with AI → Vibe Coding
- The two are complementary, not interchangeable
- Engineers often take both, in either order

## Who it's for / Format
This guide helps developers and engineers pick between Modern AI Pro's two developer-facing courses. Both are delivered as live workshops and on the self-paced subscription platform, available in the US and India.
