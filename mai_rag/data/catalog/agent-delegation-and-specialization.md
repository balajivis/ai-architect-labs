# Delegation and Specialization

Delegation and specialization is the idea that a system works better when each agent has a focused role and tasks are handed to the agent best suited for them. Like a team of experts, specialized agents outperform one generalist trying to do everything.

## Key ideas
- Specialization gives each agent a narrow, well-defined job.
- Delegation routes each task to the right specialist.
- Focused agents tend to be more reliable on their slice of work.
- This keeps prompts and tools simpler per agent.
- It's a foundation for supervisor and swarm designs.
