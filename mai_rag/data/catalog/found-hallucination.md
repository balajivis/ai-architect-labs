# Hallucination

Hallucination is when a model produces confident-sounding output that is wrong or unsupported. It is a core reliability challenge because the text can look fluent and plausible even when it is fabricated.

## Key ideas
- Models can state false information with high confidence.
- Fluency does not guarantee factual accuracy.
- Grounding answers in trusted sources reduces (but never fully removes) it.
- Verification and evaluation are essential in production.
- High-stakes uses require human review and guardrails.
