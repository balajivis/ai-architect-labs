# Consensus and Voting

Consensus and voting are ways for multiple agents to agree on an answer. Instead of trusting one agent, the system gathers several opinions and combines them to reach a more reliable result.

## Key ideas
- Several agents produce candidate answers independently.
- Voting or aggregation picks the most agreed-upon result.
- This can reduce one-off errors and improve reliability.
- Useful when correctness matters more than speed or cost.
- It trades extra computation for higher confidence.
