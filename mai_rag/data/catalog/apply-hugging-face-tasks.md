# Hugging Face Tasks

Hugging Face is a widely used hub for open machine-learning models, organized by "tasks" — categories like text classification, summarization, translation, image generation, and speech recognition. Browsing tasks helps teams discover what kinds of problems pre-built models can solve.

## Key ideas
- "Tasks" group models by what they do (e.g., classification, generation, transcription).
- Many tasks have ready-to-use open models, lowering the barrier to prototyping.
- Choosing a task is the first step in mapping a product need to a model capability.
- Open models offer alternatives to proprietary APIs, with their own trade-offs.
