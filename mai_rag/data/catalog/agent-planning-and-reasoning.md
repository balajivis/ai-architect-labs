# Planning and Reasoning

Planning and reasoning is how an agent turns a high-level goal into a sensible sequence of steps. Instead of reacting one move at a time, a planning agent thinks ahead about what needs to happen and in what order.

## Key ideas
- Reasoning breaks a vague request into clear sub-goals.
- Planning orders those sub-goals into a workable sequence.
- Plans can be made up front or revised as new information arrives.
- Good planning reduces wasted tool calls and dead ends.
- Plans should stay flexible so the agent can adapt when steps fail.
