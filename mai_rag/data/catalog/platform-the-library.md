# The Library
The Library is Modern AI Pro's reference hub — a curated home for everything learners need to look up, review, and reinforce. It brings modules, definitions, tools, and quick references together so knowledge stays accessible long after a lesson ends.

## What it is
- Course modules organized for easy review and revisiting
- A glossary and mindmaps that map concepts and their connections
- A tools directory pointing to the AI tools that matter
- Cheat sheets for fast, practical reference
- A persistent knowledge base supporting all courses

## Who it's for / Format
The Library serves every Modern AI Pro learner who wants to reinforce learning and find references quickly. It is part of the self-paced subscription platform, available to learners in the US and India.
