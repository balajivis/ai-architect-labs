# Agentic Reasoning Patterns

Agentic reasoning patterns are reusable ways of structuring how an agent thinks and acts. Rather than a single prompt, they give the agent a repeatable shape for breaking down problems, choosing actions, and improving its own output.

## Key ideas
- Common patterns include reasoning-then-acting, planning, reflection, and self-critique.
- Reflection lets an agent review and revise its own draft before finishing.
- Planning patterns split a big goal into ordered, smaller steps.
- Tool-use patterns interleave thinking with calls to external systems.
- Patterns can be combined; the right mix depends on the task's difficulty.
