# Tokens (Input / Output)

Tokens are the small chunks of text that models read and produce; a word may be one or several tokens. Both the text you send (input) and the text the model returns (output) are measured in tokens, which drives cost and length limits.

## Key ideas
- Models process text as tokens, not whole words.
- Input and output tokens are usually billed separately.
- Token counts determine cost and fit within the context window.
- Different languages and formats tokenize differently.
- Estimating tokens helps predict cost and length.
