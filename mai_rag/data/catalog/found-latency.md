# Latency

Latency is how long a user waits for a model to respond. It shapes the feel of an AI product and is often as important as raw answer quality for user experience.

## Key ideas
- Latency is the delay between request and response.
- Larger or reasoning-heavy models tend to be slower.
- Streaming responses can make perceived latency feel lower.
- It trades off against capability and cost.
- Good products budget latency the way they budget money.
