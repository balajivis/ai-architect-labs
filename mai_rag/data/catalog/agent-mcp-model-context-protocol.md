# MCP (Model Context Protocol)

The Model Context Protocol (MCP) is an open standard for connecting AI agents to external tools and data sources in a consistent way. Instead of custom integrations for every service, MCP gives agents a common plug-in interface.

## Key ideas
- MCP standardizes how agents discover and use external tools and data.
- It reduces one-off, bespoke integrations between models and services.
- Tools and data sources are exposed through MCP servers.
- An open standard means broad, reusable interoperability.
- It helps agents safely reach the systems they need.
